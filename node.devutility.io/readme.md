# node.devutility.io

## Install

``` bash
npm install utilities-io
```

## Usage

### Version 1.1.x

``` javascript
// Import default module
import IOUtilities from "utilities-io";

// Import named module
import { IOUtilities } from "utilities-io";
```

### Version 1.0.x

``` javascript
const ioUtilities = require("utilities-io");
```
