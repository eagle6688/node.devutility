# node.devutility.collection

## Install

``` bash
npm install utilities-collection
```

## Usage

### Version 1.1.x

``` javascript
// Import default module
import CollectionUtilities from "utilities-collection";

// Import named module
import { CollectionUtilities } from "utilities-collection";
```

### Version 1.0.x

``` javascript
const CollectionUtilities = require("utilities-collection");
```
