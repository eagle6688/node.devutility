# node.devutility.script

## Install

``` bash
npm install utilities-script
```

## Usage

### Version 1.1.x

``` javascript
// Import default module
import scriptUtilities from "utilities-script";

// Import named module
import { scriptUtilities } from "utilities-script";
```

### Version 1.0.x

``` javascript
const scriptUtilities = require("utilities-script");
```
